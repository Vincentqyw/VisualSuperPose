# VisualSuperPose

A tool for localizing and finding the camera pose of query images in respect to a globe-spanning database.

Developed during the class a practical course at *Karlsruhe Institute of Technology* from October 2020 to February 2021. 

This project was developed by:

* [Julian Carius](https://gitlab.com/JCarius)

* [Yin Wu](https://gitlab.com/yinwu.life)
 
This project was supervised by:  

* Thomas Pollok  
* Andreas Specker  

## Dependencies
* C++14 compiler
* CMake 3.1+
* Qt 5
* Opencv 4
* [FBoW](https://github.com/rmsalinas/fbow)
* [libtorch](https://pytorch.org/)
